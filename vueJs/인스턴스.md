## 인스턴스
- 뷰 개발 시 필수로 생성하는 코드

```
new Vue();
```

### 생성자 함수
- 함수 이름이 대문자로 시작

```
function Person(name, job){
    this.name = name;
    this.job = job;
}

new Person('josh', 'developer');
```

- Vue 라는 이름으로 정의해 놓은 생성자 함수가 있고, 함수 안에서 미리 정의해 놓은 속성 및 함수를 가져다가 사용할 수 있다.

### 재사용 옵션

- template
