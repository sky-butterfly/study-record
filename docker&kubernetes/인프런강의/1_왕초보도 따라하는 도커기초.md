## 섹션 1. (쿠버네티스 들어가기 앞서) 왕초보도 따라하는 도커기초

### 내가 원하는 이미지 찾기 도커 레지스트리
- 도커허브
    - 경로가 앞에 있으면 사용자가 업로드한 것
    - 경로가 없는 경우 도커에서 공식적으로 지원
        > 좀 더 검증이 되어 안정적으로 사용할 수 있을 것이라 생각
    - sudo docker search ** 명령어로도 찾을 수 있음
    - 누구나 다운 받을 수 있지만, 자산 유출의 문제가 있기 때문에 private 한 공간 생성도 가능
        > 도커 아이디 아직 안 만들었음...

### 도커 라이프 사이클 이해하기
- 레지스트리에서 이미지를 pull 받음
- 이미지로 컨테이너를 생성
- 컨테이너를 실행시키면 메모리에 올라감
- run 명령어로 pull, create, start 한번에 가능
