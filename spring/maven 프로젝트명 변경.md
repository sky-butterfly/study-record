## Maven 프로젝트 이름 변경

1. 프로젝트명 변경
    - 프로젝트 우클릭 -> Refactor -> Rename 에서 이름 변경

2. Artifact Id 변경
    - 프로젝트 우클릭 -> Refactor -> Rename Maven Artifact 에서 Artifact Id 변경

3. pom.xml 변경
    - pom.xml 파일의 artifactId, name 변경

4. Context root 변경
    - 프로젝트 우클릭 -> Properties -> Web Project Settings 에서 Context root 변경
